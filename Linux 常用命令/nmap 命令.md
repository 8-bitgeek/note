### 1. 基本用法: 

```shell
nmap {serverNames/ips}    # 可以写多个域名或 ip
```

### 2. 使用最后一个字节指定多台服务器

```shelL
namp 192.168.192.1,2,3
```

### 3. 指定扫描范围

```shell
nmap 192.168.192.1-100
```

### 4. 扫描操作系统信息和路由跟踪

```shell
nmap -A 192.168.192.1
```

### 5. 找出在线的主机

```shell
nmap -sP 192.168.192.*
```

### 6. 扫描指定的端口

```shell
nmap -p 8888,80 192.168.192.1     # 拂拭
nmap -p T:8888,80
```
